# Familycoin
Simple telegram bot family finance analytic 💸 📈 📉

This bot 👾 can:
- record income and expenses in the database 📈📉
- arrange data by categories 📌
- show summary reports 📝
- send notifications to your family about recent transactions 👨‍👩‍👧
